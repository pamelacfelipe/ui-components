# ui-components

Components for study/work purpose.

## ui-form-creator

```html
<ui-form-creator ng-model="vm.form" view-mode></ui-form-creator>
```

**Field format**

* Text (default) 
* Datetime
* Value
* Radio button
* Checkbox

**Dependencies**

["angular-gridster": "^0.13.14"](https://github.com/ManifestWebDesign/angular-gridster)
